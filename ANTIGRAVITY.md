# カスタム・ポモドーロタイマー

## 1. 概要
「1分間の準備（アイドリング）」から始まる、ADHD特性に配慮したシングルページWebアプリケーション。
作業開始の心理的障壁を下げることと、集中を妨げない安定したUIを提供することを最優先としている。

## 2. 目的・ビジョン
- **着火のサポート**: 「1分だけやる」というアイドリング機能により、作業開始をスムーズにする。
- **視覚的安定性**: UIのガタつき（レイアウトシフト）を徹底的に排除し、視覚的なノイズを最小限に抑える。
- **シームレスな記録**: タスク履歴のサジェストとCSV出力により、自身の活動を簡単に振り返れるようにする。

## 3. 技術スタック
- **Frontend**: HTML5, CSS3 (Vanilla), JavaScript (Vanilla ES6+)
- **Storage**: Browser localStorage（設定・タスク履歴の保存）
- **Deployment**: GitHub Pages
- **Assets**: Google Fonts (Inter, Outfit), Web Audio API (通知音生成)

## 4. 主要機能
- [x] **アイドリングモード**: 作業開始前の1分間のカウントダウン。終了後は自動で作業モードへ。
- [x] **無制限タイマー**: 作業・休憩時間を秒単位でカウントアップ計測。
- [x] **タスク管理UI**:
    - ブラウザ標準を廃止したカスタムドロップダウンサジェスト。
    - 入力中も全履歴を表示可能なカスタム矢印アイコン。
    - 履歴の個別削除機能。
- [x] **データ永続化**: 通知音、CSV出力設定、タスク履歴をブラウザに保存。
- [x] **CSVエクスポート**: セッション履歴をBOM付きUTF-8で出力。
- [x] **プレミアム・サウンド**: Web Audio APIによる「やる気が出る」上昇アルペジオ通知音。

## 5. デザイン原則
- **グラスモルフィズム**: 背景が透過し、奥行きを感じさせるモダンな質感。
- **完璧なセンタリング**: 入力欄のパディングを左右対称に固定し、テキストの中心軸が常に画面中央に来るよう調整。
- **キャッシュ対策**: GitHub Pages展開時の反映遅延を防ぐキャッシュバスター（`?v=1.1`）の導入。

## 6. 現在のステップ
- **Step 5 (完了)**: データ管理の強化。CSV手動ダウンロードボタンと履歴リセット機能の実装、および終了時の挙動変更。
- **現状**: 「今日の作業終了」後でも履歴が保持され、設定パネルからいつでもCSV保存やリセットが可能な、より柔軟なデータ管理体制が整った。
